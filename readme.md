# GeoFlux (P5 Visualizer)

React と p5.js を組み合わせた、高性能なジェネレーティブアート・ビジュアライザーです。
幾何学的なアルゴリズムやモアレ干渉、GLSLシェーダーを用いて、美しく複雑なパターンをリアルタイムに生成します。
オーディオリアクティブ機能を搭載しており、音楽や環境音に合わせてビジュアルが有機的に変化します。

## ✨ 主な機能 (Features)

### 🎨 多彩な描画モード
*   **Moire (モアレ)**: 円形の重なりによる干渉縞の美しさを表現。
*   **Spiral (螺旋)**: 回転する螺旋状のパーティクルシステム。
*   **Grid (グリッド)**: 回転差を利用したグリッドの干渉。
*   **Random (ランダム)**: 星屑のように点と線を動的に接続。
*   **Flower (フラワー)**: 花弁のような放射状の幾何学パターン。
*   **Wave (ウェーブ)**: 複数のサイン波を重ね合わせた波紋。
*   **Shader (シェーダー)**: GLSLを用いた高度なピクセルシェーダー描画。

### 🎵 オーディオリアクティブ
*   **音声入力**: マイク入力またはローカルの音声ファイル(.mp3, .wav等)に対応。
*   **周波数解析**: 低音(Bass)、中音(Mid)、高音(Treble)それぞれの帯域に対する感度を個別に調整可能。
*   ビジュアルのサイズ、色、シェーダーのパラメータが音に反応して躍動します。

### 🛠 Pro Lab (カスタムモード)
*   **実験室モード**: 通常のプリセットとは独立した「Proモード」を搭載。
*   リサージュ曲線やフィードバック変調のパラメータを細かく制御し、独自の「カオス・エンジン」を用いた新しいパターンを発見できます。

### 💾 エクスポート & ツール
*   **高解像度画像保存**: 生成されたアートをPNG形式で保存可能。
*   **ワンライナー生成**: 現在のパラメータ設定（一部モードを除く）を、Twitter(X)などで共有可能な短い `p5.js` コードに変換してコピーできます。
*   **ポストプロセス**: ミラー（万華鏡）、色反転、グリッチエフェクトを搭載。
*   **ズーム機能**: プレビュー領域の拡大縮小が可能。

## 🚀 使い方 (Usage)

1.  **基本操作**:
    *   画面右側のパネルで「パターンモード」や「カラーパレット」を選択します。
    *   各スライダーを操作して、速度、透明度、形状の密度などを調整します。

2.  **音楽と同期**:
    *   「🎵 音声リアクティブ」セクションで音声ファイルをアップロードするか、再生ボタンを押します（マイク/ファイル）。
    *   感度スライダーで反応の強さを調整してください。

3.  **Proモード**:
    *   「🛠 Proモード (カスタムラボ) ON」ボタンをクリックすると、左側に詳細設定パネルが開きます。
    *   ここでは、ベースとなる形状の周波数や変調度合いを深く弄ることができます。

4.  **保存**:
    *   「📷 画像保存」ボタンでスクリーンショットを保存。
    *   「📋 ワンライナーコードをコピー」でコードを取得。

## 💻 技術スタック

*   **Frontend Library**: React 19
*   **Language**: TypeScript
*   **Creative Coding**: p5.js, p5.sound
*   **Styling**: CSS (Responsive Design)

## ⚠️ 注意事項

*   **ブラウザ互換性**: モダンなブラウザ（Chrome, Firefox, Edge, Safari）での動作を推奨します。
*   **パフォーマンス**: パーティクル数や解像度を上げすぎると、デバイスによっては動作が重くなる場合があります。
*   **音声権限**: オーディオリアクティブ機能を使用する際は、ブラウザのマイクアクセス許可が必要です。
